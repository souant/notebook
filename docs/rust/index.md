# rust

[《Rust 程序设计语言》](https://rustwiki.org/zh-CN/book/title-page.html)

查看 Rust 编译器

```shell
rustc --version
```

编译代码

```shell
rustc ./main.rs
```

Rust 项目构建及包管理工具 Cargo

```shell
cargo --version
```

1. 新建

   ```shell
      cargo new project_name
   ```
2. 检查

   ```shell
   cargo check
   ```
3. 运行

   ```shell
   cargo run
   ```
4. 构建

   ```shell
   cargo build
   ```
5. 发布

   ```shell
   cargo build --release
   ```
